A blogging Website where users can -->
    Sign in, Sign out, create, update and delete posts.

A simple blogging platform built with the following stack:

- **Frontend:** React.js  
- **Backend:** Hono (Node.js framework)  
- **Database:** PostgreSQL  
- **ORM:** Prisma  
- **Authentication:** JWT  
- **Validation:** Zod  
- **Deployment:** Vercel + Supabase
